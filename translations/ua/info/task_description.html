p> Наші Роботи ніколи не упускають шансу поліпшити свої навички у лінгвістиці. Зараз вони вивчають англійський алфавіт і те, для чого він створений.

Алфавіт розділений на голосні та приголосні букви (так, ми розділили букви, а не звуки).
Голосні -- A E I O U Y
Приголосні -- B C D F G H J K L M N P Q R S T V W X Z

Дано текст з різними словами та/або числами, які розділені пробілами та знаками пунктуації. Числа не вважаються за слова (також як і суміш букв та цифр). Необхідно підрахувати слова, в яких голосні букви чергуються із приголосними (чергуючі). Тобто в таких словах немає двох голосних або двох приголосних букв підряд. Слова, які складаються із однієї букви - не рахуються. Регістр букв не має значення.

Вхідні дані: Текст, як рядок (str).

Вихідні дані: Кількість "чергуючих" слів, як ціле число (int).

Приклади: 

checkio("My name is ...") == 3
checkio("Hello world") == 0
checkio("A quantity of striped words.") == 1, "Only of"
checkio("Dog,cat,mouse,bird.Human.") == 3

Навіщо це потрібно: Концепції, що використовуються у даній задачі можуть бути використаними для лінгвістичного аналізу. Обробка тексту – дуже важливий інструмент для аналізу творів та мов.

Передумови:Текст містить лише ASCII символи.
